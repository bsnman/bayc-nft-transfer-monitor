# Setup

## Prerequisite

python 3.12
poetry 1.8.2

## Env

Create `.env` file and copy the content of `.env.example` into it

## Installation

Install the packages using poetry

```
poetry install
```

Then run the virtual environment with

```
poetry shell
```
